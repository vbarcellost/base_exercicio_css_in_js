# EBAC Jobs

Projeto desenvolvido em React e TypeScript para apresentar e pesquisar vagas nas áreas de tecnologia, design e criatividade.

Neste exercício, toda a estilização original foi convertida para **Styled Components**, substituindo os arquivos CSS e CSS Modules por uma abordagem CSS-in-JS.

## Tecnologias utilizadas

- React
- TypeScript
- Styled Components
- Create React App

## Componentes estilizados

- formulário de pesquisa;
- cabeçalho;
- hero;
- componente `Vaga`;
- componente `ListaVagas`;
- estilos globais e responsivos.

## Como executar o projeto

Depois de baixar o projeto, instale as dependências:

```bash
npm install
```

Inicie o servidor de desenvolvimento:

```bash
npm start
```

A aplicação será aberta em [http://localhost:3000](http://localhost:3000).

## Comandos disponíveis

### `npm start`

Executa a aplicação em modo de desenvolvimento. A página será atualizada automaticamente quando o código for alterado.

### `npm test`

Executa os testes em modo interativo.

### `npm run build`

Cria uma versão otimizada da aplicação para produção na pasta `build`.

## Funcionalidades

- visualização das vagas disponíveis;
- pesquisa por cargo ou tecnologia;
- contador de resultados;
- mensagem para pesquisas sem resultados;
- layout adaptado para computadores, tablets e celulares.
